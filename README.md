# 🧠 뷰티파크 세컨브레인 (LLM Wiki · 3층 구조)

피부·미용 의료 지식 · 마케팅 · 경영을 위한 AI 세컨드 브레인 볼트입니다. 안드레이 카파시의 'LLM 위키' 3층 아키텍처로 구축되었습니다.

## 구조

- **1층** `01_Raw_Data/` — 날것의 원본 창고 (AI는 읽기만). 새 자료는 `_INBOX` 에.
- **2층** `02_AI_Wiki/` — AI가 관리하는 단 하나의 진실(SSOT). 시작점: `02_AI_Wiki/00_MOC_지도/00_홈_MOC.md`.
- **3층** `CLAUDE.md` — AI 사서 행동 규칙서.

```
뷰티파크-obsidian/
├─ CLAUDE.md                    ← 3층: AI 사서 규칙서
├─ 01_Raw_Data/                 ← 1층: 원본 창고 (READ-ONLY)
│   ├─ _INBOX/                  ← 새 자료 투입구
│   └─ 5개 팀 폴더 (마케팅·세일즈상담·교육운영·콘텐츠대본·의학지식)
├─ 02_AI_Wiki/                  ← 2층: AI가 관리하는 SSOT
│   ├─ 00_MOC_지도/00_홈_MOC    ← 시작점
│   ├─ 01_마케팅    (SOP·채널성과·의료광고법가이드)
│   ├─ 02_세일즈상담 (상담스크립트·환불규정·페르소나)
│   ├─ 03_교육운영   (운영SOP·온보딩)
│   ├─ 04_콘텐츠대본 (VSL공식·기획보드)
│   ├─ 05_의학지식   (시술핵심지식·진료프로토콜·팩트체크로그)
│   ├─ 06_경영      (대시보드·KPI)
│   └─ 99_Insights
└─ .claude/agents/              ← AI 직원 7명
```

## 🏢 AI 사무실 (픽셀 오피스 UI)

`ai-office/index.html` 을 브라우저로 열면 AI 직원 7명이 있는 픽셀 사무실이 뜹니다. 직원을 클릭해 업무를 지시하거나 가르칠 수 있습니다. 자세한 사용법은 `ai-office/README.md` 참고.

## 지금 시작하기

1. 이 폴더를 **옵시디언 볼트로 열기** (Obsidian → 폴더를 볼트로 열기).
2. `00_홈_MOC` 노트에서 전체 지도 확인.
3. 원본 자료를 `01_Raw_Data/_INBOX` 에 넣고 AI 사서에게 **"넣기 해줘"** 요청.
