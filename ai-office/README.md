# 🏢 뷰티파크 AI 사무실

옵시디언 세컨브레인의 AI 직원 7명을 픽셀 사무실로 시각화한 프론트데스크입니다. (마하닥 AI 사무실 스타일)

## 실행

`index.html` 을 브라우저로 열면 끝. 빌드·설치 불필요.

```bash
# 또는 로컬 서버로
python3 -m http.server 8000   # → http://localhost:8000/ai-office/
```

## 사용법

1. 픽셀 사무실에서 **직원 클릭** → 오른쪽 패널에 프로필·탭(업무 / 직무기술서 / 기억·교훈 / 업무 일지)이 뜹니다.
2. 아래 입력창에서 **🎯 업무 지시** 또는 **🎓 가르치기(피드백)** 를 선택하고 전달.
   - 가르친 내용은 그 직원의 **기억·교훈**에 저장되고, 이후 모든 업무에 자동 반영됩니다.
3. 직원이 실제로 일하게 하는 두 가지 모드:
   - **API 모드**: 오른쪽 위 `⚙ API 키` 에 **Anthropic 키(`sk-ant-…`) 또는 Google Gemini 키(`AIza…`)** 입력 — 키 앞자리로 자동 판별해서 Claude 또는 Gemini가 이 화면에서 바로 답합니다.
   - **Claude Code 모드**: 업무 카드의 `프롬프트 복사` → 볼트 루트에서 Claude Code에 붙여넣기 → `.claude/agents/` 의 해당 직원 에이전트가 위키를 읽고 쓰며 진짜 업무를 수행합니다.

## 데이터

업무 이력·기억·일지는 전부 브라우저 localStorage에 저장됩니다 (서버 없음). API 키도 localStorage에만 저장되므로 공용 PC에서는 사용 후 비우세요.
