# 구매 퍼널 (이커머스 MOT)

한 줄: **고객이 재생크림을 만나 사기까지의 모든 "진실의 순간"을 지도로 그려, 어디서 새는지 찾는다.**

## 퍼널

```
노출(광고·검색·SNS) → 유입(상세페이지 방문) → 장바구니
→ 결제 → 수령·사용 → 재구매·리뷰·추천
```

## 단계별 관리 포인트

| 단계 | 담당 지식 | 이탈 시 확인 |
|---|---|---|
| 노출→유입 | [[네이버쇼핑_스마트스토어_최적화]], [[GEO_상품페이지_최적화]] | 노출은 되는데 클릭이 없나? (썸네일·상품명) |
| 유입→장바구니 | [[상세페이지_공식]] | 상세는 보는데 안 담나? (후킹·성분 설득) |
| 장바구니→결제 | 배송비·결제 UX·쿠폰 | 담고 이탈? (리마인드·혜택) |
| 결제→재구매 | [[재구매_리텐션]] | 한 번 사고 안 오나? (소진주기 CRM) |

## 활용

- 각 단계 전환율을 숫자로 → 가장 많이 새는 곳 하나부터 개선 ([[KPI_대시보드]])
- 광고비 배분은 노출→유입 실측(ROAS)에 근거 ([[채널별_성과]])
