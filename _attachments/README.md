# 📎 _attachments — 이미지·첨부 보관함

옵시디언에서 노트에 **사진을 붙여넣기(Ctrl/Cmd+V)하거나 드래그·업로드**하면 이 폴더에 자동 저장되고, 노트에는 `![[파일명.png]]` 형태로 삽입됩니다. (설정: `.obsidian/app.json`의 `attachmentFolderPath`)

## ⚠️ 중요 — 사진은 git에 커밋되지 않습니다

이 저장소는 **공개(public)** 이므로, 시술 전후 사진·환자 자료 등이 실수로 공개되지 않도록 이미지·영상 파일은 `.gitignore`로 커밋에서 제외돼 있습니다. 즉:

- 붙여넣은 사진은 **이 기기(로컬 볼트)에만** 저장됩니다. 노트를 열면 잘 보입니다.
- 하지만 git으로 다른 기기와 **자동 동기화되지는 않습니다.**

## 기기 간에 사진도 동기화하고 싶다면

1. **먼저 GitHub 저장소를 Private으로 전환** (Settings → General → Danger Zone → Change visibility). — 의료 볼트는 Private을 강력 권장합니다.
2. 그 뒤 `.gitignore`에서 이미지 제외 줄을 빼면 사진도 커밋·동기화됩니다. (그래도 `.vercelignore` 덕분에 배포 사이트에는 노출되지 않습니다.)

## 개인정보 (CLAUDE.md §3-5)
