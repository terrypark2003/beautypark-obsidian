# 네이버 플레이스 최적화

한 줄: **국내 병원 신환 유입의 최전선. 순위는 N지수(특히 N2 총점수)로 움직인다.**

> 출처: 마하닥AI 경영 분석기 방법론 ([[마하닥AI_경영분석기_기능분석]]). 네이버 알고리즘은 비공개이므로 아래 지수는 관측 기반 추정 모델임 — 절대 수치가 아니라 방향으로 활용.

## N지수 (플레이스 순위를 만드는 3요소)

| 지수 | 의미 | 우리가 할 일 |
|---|---|---|
| **N1** 세팅 점수 | 기본 정보·사진·업체 세팅 완성도 | 정보 100% 채우기, 사진 다량·정기 업로드 |
| **N2 ★** 총점수 | 방문자·블로그·영수증 **리뷰 전체** 활동 | **가장 중요** — 리뷰 수·최신성 관리 |
| **N3** 경쟁 점수 | 키워드별 경쟁 강도 | 경쟁 약한 키워드부터 공략 |

**핵심: 순위를 올리는 가장 강한 레버는 N2(리뷰 활동)다.** 세팅(N1)은 기본기, 리뷰(N2)가 승부처.

## 플레이스지수 등급

| 등급 | 지수 | 상태 |
|---|---|---|
| S | 0.30 ↑ | 최상위 |
| A | 0.27~0.30 | 우수 |
| B | 0.25~0.27 | 보통 |
| C | 0.23~0.25 | 개선 필요 |
| D | 0.23 ↓ | 취약 |

## 실행 체크리스트

- [ ] 15개 기본 진단 항목 점검 (정보 완성도·사진 수·영업시간·소식 등)
- [ ] 대표 키워드 선정: **반드시 지역명 포함**(플레이스는 지역 기반 검색 노출)
- [ ] 리뷰 3종(방문자·블로그·영수증) 유입 루틴 → 상담·CS와 연계 ([[운영_SOP]])
- [ ] 경쟁 병원 대비 대표키워드 공통 분석 → 빈틈 키워드 확보
- [ ] 순위 추적: 키워드별 주간 기록 → [[채널별_성과]]
