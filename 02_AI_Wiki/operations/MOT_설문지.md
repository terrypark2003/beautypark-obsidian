# MOT 설문지 (내원경로 조사)

한 줄: **신환에게 "어떻게 알고 오셨어요?"를 표준 양식으로 물어, [[MOT_내원경로_퍼널]]의 인지 채널을 실측한다.**

> 출처: 마하닥AI 경영 분석기 MOT설문지 기능 부록 ([[마하닥AI_경영분석기_기능분석]]).

## 데스크 신환 설문 (예시 항목)

1. **어떻게 알게 되셨나요?** (복수응답): 네이버 검색 / 플레이스 / 블로그 / 유튜브·인스타 / 지인 소개 / 간판·지나가다 / 기타
2. **결정에 가장 영향을 준 것은?** (택1)
3. **거주지(동 단위)** → [[진료권_분석]] 검증용
4. **가장 신경 쓰이는 고민** → [[conditions_MOC]] 매핑
5. (선택) 다른 병원과 비교하셨나요?

## 운영 방법

- 데스크 태블릿/QR 또는 종이 → 월 단위 집계
- 결과는 [[채널별_성과]]의 "인지 채널" 실측값으로 반영
- ⚠️ 개인정보 최소 수집·익명 집계 (CLAUDE.md §3-4)

## 활용

- 광고비를 어느 채널에 쓸지 = 이 설문의 인지 채널 순위가 근거
- [[MOT_내원경로_퍼널]]의 "인지→관심" 단계 진단
