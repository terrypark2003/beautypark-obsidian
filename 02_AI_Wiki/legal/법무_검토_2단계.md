# 법무 검토 2단계 체계

한 줄: **모든 사안은 1차 통합검토관을 거친다. 중요도 낮으면 1차에서 끝, 높으면 영역별 2차 전문검토관으로.**

## 흐름

```
사안 → [1차 통합검토관 legal-officer]
        ├─ 중요도 낮음 → 1차에서 종결 (의료법·노무·세무 전 영역 커버)
        └─ 중요도 높음 → 영역별 2차 에스컬레이션
                          ├─ 의료법 → legal-medical
                          ├─ 노무   → legal-labor
                          └─ 세무   → legal-tax
```

## 검토자

| 단계 | 직원 | 담당 |
|---|---|---|
| 1차 | ⚖️ [[법령_조회_가이드|법무검토관(1차)]] `legal-officer` | 전 영역 1차 검토·분류·중요도 판정, 저중요 종결 |
| 2차 | ⚖️🩺 의료법 `legal-medical` | 의료법·의료광고법·개인정보 |
| 2차 | ⚖️👥 노무 `legal-labor` | 근로기준법·인사·임금 |
| 2차 | ⚖️💰 세무 `legal-tax` | 부가세·소득세·비용처리 |

## 중요도 판정 기준

**낮음 (1차 종결)**
- 내부 참고용 정리, 일반 절차 안내
- 위험이 명백히 낮은 표현·문구 확인
- 단순 법령 조항 위치·정의 조회

**높음 (2차 에스컬레이션 — 애매하면 무조건 2차)**
- 외부로 나가는 광고·콘텐츠(의료광고 심의 대상)
- 채용·해고·징계·근로계약·임금·근로시간(노무)
- 세금 신고·비용 처리·사업 구조 변경 등 금액 큰 사안
- 벌칙·과태료·행정처분·분쟁·개인정보 문제 가능성
- 조문 해석이 갈리거나 최신 개정 확인 필요

## 공통 철칙

- 조문은 [[법령_조회_가이드]]로 **원문 조회 후 인용**(조·항·호 + 시행일).
- 어느 단계든 **최종 법률 자문 아님** — 리스크 크면 변호사·노무사·세무사·심의기구 확인 명시.
