# 법령 조회 가이드 — 국가법령정보 오픈API

한 줄: **법령 조항을 인용하기 전에 이 API로 원문을 조회한다.** (법무팀 에이전트 전용 도구)

## 0. 사전 준비 — OC 키 (1회, 무료. 대표님이 하실 유일한 단계)

1. [open.law.go.kr](https://open.law.go.kr) 접속 → 로그인 → **OPEN API → 활용 신청**(자동 승인, 몇 분).
2. 발급되는 **OC 값** = 보통 가입 이메일의 **아이디 부분**(예: `hong@x.com` → `hong`).
3. 그 값을 볼트의 **`.claude/law-api.local.txt`** 파일에 한 줄로 저장한다.
   - 이 파일은 `.gitignore`로 **공개 저장소에 올라가지 않는다**(키 노출 방지).
   - 템플릿: `.claude/law-api.example.txt` 참고.
4. 이제 아래 호출은 그 파일에서 OC를 자동으로 읽는다 — 매번 입력할 필요 없음.

```bash
# 에이전트는 OC를 로컬 파일에서 읽는다 (없으면 안내 후 폴백)
OC=$(cat .claude/law-api.local.txt 2>/dev/null | tr -d '[:space:]')
```

## 1. 법령 검색 (목록에서 법령 ID·MST 찾기)

```bash
# 의료법 검색 (JSON)
curl -s "https://www.law.go.kr/DRF/lawSearch.do?OC=$OC&target=law&type=JSON&query=의료법"
```

- 주요 파라미터: `OC`(필수), `target=law`, `type=JSON|XML|HTML`, `query=검색어`, `display`(개수), `page`.
- 응답에서 **법령ID(`법령ID`)** 또는 **법령일련번호(`MST`)**, 시행일자를 확보한다.

## 2. 법령 본문·조문 조회

```bash
# 법령 전체 본문 (MST 또는 ID 사용)
curl -s "https://www.law.go.kr/DRF/lawService.do?OC=$OC&target=law&type=JSON&MST=<법령일련번호>"

# 특정 조문만 (예: 의료법 제56조) — JO에 6자리(조4+항2) 형식
curl -s "https://www.law.go.kr/DRF/lawService.do?OC=$OC&target=law&type=JSON&MST=<MST>&JO=005600"
```

- `JO`: 조문번호 6자리(앞 4자리 조, 뒤 2자리 항). 예) 제56조 → `005600`.
- 시행일·개정 이력은 본문 응답의 시행일자 필드로 확인.
- `.claude/law-api.local.txt`가 비어 있으면(`$OC` 없음) 호출하지 말고, [law.go.kr](https://www.law.go.kr) 링크·검색어로 폴백하고 대표님께 OC 키 발급을 안내한다.

## 3. 자주 쓰는 법령 (검토자별 조회 시작점)

**의료법 2차(legal-medical)**
| 법령 | 왜 |
|---|---|
| 의료법 (특히 §56 의료광고 금지·제한) | 외부 콘텐츠·광고 검수의 핵심 → [[의료광고법_가이드]] |
| 의료법 시행령 / 시행규칙 | 광고 심의·세부기준 |
| 개인정보 보호법 | 상담 녹취·CS·전후사진 (CLAUDE.md §3-4) |
| 표시·광고의 공정화에 관한 법률 | 과장·기만 광고 |
| 화장품법 / 시행규칙 | (슬로보다 연계) 표시·광고, 기능성 표방 |

**노무 2차(legal-labor)**
| 법령 | 왜 |
|---|---|
| 근로기준법 | 근로계약·임금·근로시간·해고·연차 |
| 최저임금법 | 최저임금 준수 |
| 근로자퇴직급여 보장법 | 퇴직금 |
| 남녀고용평등법 | 차별·모성보호 |

**세무 2차(legal-tax)**
| 법령 | 왜 |
|---|---|
| 부가가치세법 | 미용목적 시술 과세/면세 구분 (핵심) |
| 소득세법 | 사업소득·원천징수·종합소득세 |
| 법인세법 | 법인 운영 시 |
| 조세특례제한법 | 세액공제·감면 |

## 4. 호출이 막힐 때

- OC 미발급·오류·네트워크 차단 시: [law.go.kr](https://www.law.go.kr) 링크와 검색어(예: "의료법 제56조")를 제시하고 사용자가 직접 확인하도록 안내.
- API 응답을 인용할 때 **조회 일자**를 함께 적는다(법령은 개정될 수 있음).
