# MOT · 내원경로 퍼널

한 줄: **환자가 우리 병원을 만나는 모든 "진실의 순간(MOT)"을 지도로 그려, 어디서 새는지 찾는다.**

> MOT = Moment of Truth. 출처: 마하닥AI 경영 분석기 방법론 ([[마하닥AI_경영분석기_기능분석]]).

## 내원경로 퍼널 (전형)

```
인지(검색·광고·소개) → 관심(플레이스·블로그·홈피 열람)
→ 문의(전화·톡·예약) → 내원(첫 방문) → 시술 결정 → 재방문·소개
```

각 화살표가 **MOT** — 여기서 환자가 다음 단계로 갈지, 이탈할지 결정된다.

## MOT별 관리 포인트

| MOT | 담당 지식 | 이탈 시 확인 |
|---|---|---|
| 인지→관심 | [[네이버_플레이스_최적화]], [[GEO_AI검색_최적화]] | 노출은 되는데 클릭이 없나? |
| 관심→문의 | [[상담_스크립트]], 플레이스 리뷰 | 봤는데 문의를 안 하나? |
| 문의→내원 | [[세일즈_플레이북]], [[가격_상담_응대_FAQ]] | 문의는 오는데 예약이 안 되나? |
| 내원→시술 | [[시술_추천_프레임워크]], [[스토리텔링_상담]] | 상담은 하는데 결정을 안 하나? |
| 시술→재방문 | [[VIP_멤버십]], 사후관리 | 한 번 오고 안 오나? |

## 활용법

1. **내원경로 설문**(신환에게 "어떻게 알고 오셨어요?")으로 인지 채널 실측 → [[채널별_성과]]
2. 퍼널 단계별 전환율을 숫자로 → 가장 많이 새는 MOT 한 곳부터 개선
3. 개선 우선순위는 [[클리닉_경영]]의 "싼 레버 먼저" 원칙 적용
