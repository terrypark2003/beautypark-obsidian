## 프로그램 개요

- 이름: **마하닥AI 경영 분석기** (마하닥 김원장, 소아청소년과 전문의)
- 성격: 병의원 원장 대상 SaaS. 구글 로그인 + 관리자 승인제.
- 구조: 좌측 메뉴(마케팅 경영 / 진료 경영) + 컨설팅 AI + 전자책 + 할일관리 + 설정.

## 기능 맵

### 마케팅 경영 — GEO
- **홈페이지 AI 최적화 분석**: robots.txt·JSON-LD·sitemap·llms.txt·Meta Tags 자동 체크 → /100점. AI 봇(ChatGPT·Claude·Perplexity) 크롤링 허용 여부 진단.
- **블로그 글 EEAT 분석**: Article 스키마·저자 정보·콘텐츠 충분성·FAQ 구조 → /100점. AI 인용 신뢰도.
- **구글 상위10 비교**

### 마케팅 경영 — 블로그
- **키워드 조회** / **블로그 분석**(노출지수 100 = 노출85 + 활동15) / **통합검색 노출** 확인.

### 마케팅 경영 — 플레이스 (네이버)
- **플레이스 분석**: URL/ID 입력 → 15개 항목 자동 진단 (사진 수 등).
- **플레이스 순위**: 키워드별 순위(page 1~3, 최대 90위, 5등 내만 정밀 측정).
- **대표키워드 추천**: 상위 5개 플레이스의 공통 키워드 분석(중복 수 = 중요도).
- **키워드 순위 점수 — N지수**:
  - N1 = 플레이스 세팅 점수(기본 정보·사진·세팅 완성도)
  - N2 ★ = 플레이스 총점수(방문자·블로그·영수증 리뷰 전체) — **가장 중요**
  - N3 = 키워드별 경쟁 점수(경쟁 강도)
- **우리병원 플레이스지수** 등급: S 0.30↑ / A 0.27~0.30 / B 0.25~0.27 / C 0.23~0.25 / D 0.23↓
- **플레이스 컨설팅** 신청.

### 진료 경영
- **내원경로 분석** / **마하닥MOT퍼널**(Moment of Truth) / **MOT컨설트** / **MOT설문지 만들기**
- **검사 해석**: CBC 면역지표(NLR·SII·SIRI·NK추정) 등 수치 기반 해석
- **음성 차팅** / **진료권 분석**(주변 아파트·인구 참조) / **온톨로지** / **인구분석** / **원가분석**
- **이번달 북리뷰**(경영/마케팅 도서 토론)

### 컨설팅 AI
- **마하닥 김원장 컨설팅AI**: 강의자료 업로드(RAG) 기반. "강의자료만(엄격)" vs "강의자료+GPT 보충" 모드.

### 전자책 라이브러리 (원장 전용)
